# Innova - Portfolio Premium

https://juuann.netlify.app/

Portfolio de alta performance desenvolvido para Seu Nome, com foco em design futurista, animações fluidas e experiência do usuário imersiva.

## Tecnologias

- **Vite**: Build tool ultra-rápida.
- **React**: Biblioteca UI para componentes interativos.
- **TypeScript**: Tipagem estática para robustez.
- **Tailwind CSS**: Estilização utilitária e design system.
- **GSAP**: Animações profissionais de alta performance.
- **Swiper**: Sliders de toque modernos.
